# OriSelf Next

[中文](./README.md)

**Product-as-skill conversational personality test.** The product IS the `SKILL.md` — any LLM runtime loads it and the full interview methodology is ready. No code required.

> "It's not a test system — it's a friend having a chat." — Through ~20 rounds of natural conversation, OriSelf delivers an MBTI type + personalized insight + a unique editorial report page — all without feeling like a test.

Most personality tests hand you 93 multiple-choice questions and a generic report. OriSelf is different: an LLM-powered interviewer that chats with you like a friend who just happens to know cognitive science, observes your language patterns, and converges on a personality profile **grounded in what you actually said** — not what a checkbox implied.

The core innovation: **the product is a set of markdown files.** `skills/oriself/SKILL.md` contains the entire interview methodology — phases, techniques, guardrails, convergence logic. Drop it into any LLM runtime that supports skills and the system works.

**Apache 2.0** — free, open source. The interview methodology is yours to fork, adapt, and build on.


Done. Restart Claude Code, say "I want to take an MBTI test". That's it.

## What Gets Delivered

At the end of ~20 rounds, OriSelf converges and delivers three things:

| Deliverable | Description |
|-------------|-------------|
| **MBTI Type** | e.g. INTJ, with per-dimension confidence scores |
| **Personalized Insight** | 3 paragraphs grounded in things you actually said — not template text |
| **Editorial Report Page** | A unique HTML page per personality type — completely different visual design, not a template swap |

The report is generated by the LLM at convergence time. No two reports look alike.

## Five Rules (when in conflict, the rules win)

1. **Mirror their voice.** They're casual, you're casual. They write 300 words, you don't reply in two sentences.
2. **Emotion first.** When they share something heavy, acknowledge the feeling before collecting data.
3. **One question per turn.** Never stack questions.
4. **Stay in the conversation.** No meta-commentary, no "great question!", no therapist-speak.
5. **Evidence must be grounded.** Every personality signal you log must quote the user's actual words.
