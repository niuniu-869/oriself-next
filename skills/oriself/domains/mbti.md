# Domain Lens · MBTI

> session.domain = "mbti" 时加载。给 LLM 提供 MBTI 四维的题池和 "从回答中识别信号" 的指引。v2.4 · 这里不是契约，是参考。

## 维度定义（给 LLM 参考，不要说给用户听）

### E / I · 能量朝向
- **E**：与人共处后 TA 感到被充电；独处一天后有点闷
- **I**：社交后需要独处恢复；一天不说话也 OK

### S / N · 信息摄取
- **S**：注意具体细节、过往经验、当下的画面
- **N**：跳到模式、意义、未来可能性

### T / F · 决策方式
- **T**：按逻辑 / 标准 / 对错决定
- **F**：按影响人 / 关系 / 价值观决定

### J / P · 生活节奏
- **J**：喜欢有计划、名单、结束感
- **P**：开放式、临时起意、讨厌关门

## 情境题池（按维度）

**每轮只从一个维度挑一道**。相邻 3 轮尽量换维度。

### E/I 情境题

1. "上周一个完整没出门的一天，过到晚上你感觉怎么样？"
   —— TA 愿意讲周末细节时最灵
2. "最近一次参加超过 10 个人的聚会，结束后回家路上你第一反应是什么？"
   —— ★ **E/I 最稳的能量反馈题**。TA 刚讲完一次社交场景时顺势问最自然
3. "你认识一个新朋友，你是倾向于第 2-3 次见面前就主动发消息，还是等 TA 先发？"
   —— 行为题。TA 讲完一段关系时用
4. "周末如果有 6 个小时空出来，你的默认剧本是什么？（不用想什么'应该'，想默认）"
   —— TA 愿意多讲一点时用。别问累着的人"理想周末"
5. "你上一次觉得'这个人懂我'是跟谁在一起？你们那天具体做了什么？"
   —— 带画面、带关系，能顺带套出 T/F 的信号
6. "最近一周，一个人吃饭几次？和人吃饭几次？哪个更舒服？"
   —— ★ **TA 话短但还在聊时最灵**。成本低、画面大、答案直接指向能量

### S/N 情境题

1. "给我讲一个最近 2 周你记得最清楚的一个生活细节 —— 不用是大事，可以是'那天早上咖啡很烫'这种。"
   —— ★ **TA 开始用比喻 / 抽象化时回拉到这题**。把意象落到"具体哪天、哪条街"
2. "你如果现在想起高考那年，第一个蹦出来的画面是什么？是一个具体场景还是一种感觉？"
   —— 看 TA 答的是画面还是感觉，这题本身就在区分 S/N
3. "最近你有没有'突然想到一个不相关但有趣的联系'？比如看到 A 想到 B，这两件事看起来没关系？"
   —— N 的反向验证题。TA 能自发举出就是强 N 信号
4. "你读一本书，更容易被'作者怎么讲这个故事'打动，还是'这个故事本身'打动？"
   —— 文艺气质的被试适用
5. "昨天晚餐吃了什么？—— 如果你记得细节，形容一下那顿饭。"
   —— ★ **TA 话短时最灵**。记得清是 S，记不清但能讲出餐桌氛围是 N
6. "你是那种会在头脑里'看到'五年后的自己的人，还是更习惯只想'下周'？"
   —— 和 TA 当下的规划话题挂钩时顺势问

### T/F 情境题

1. "最近一次你帮朋友做决定 / 给建议。你是怎么想的？你说了什么？"
   —— ★ T/F 主力题。TA 讲过朋友时自然接
2. "你和一个朋友因为同一件事意见不同，你倾向于'把道理说明白'，还是'先理解 TA 为什么这么想'？"
   —— 抽象一点，适合 TA 愿意思考时用
3. "你下过一个你后来觉得'理性来说对，但我心里不舒服'的决定吗？讲讲那次。"
   —— 敢说的被试用。能同时看到 T/F 两面
4. "你看到别人哭，你第一反应是'想帮 TA 想办法'还是'想坐在旁边陪着'？"
   —— ★ **TA 话短时最灵**。二选一、有画面、成本最小
5. "最近一次你拒绝了某个人 / 某件事。讲讲那个场景，你是怎么说的拒绝的话？"
   —— 看 TA 怎么拒的，能同时抽 T/F 和 J/P
6. "如果一个项目必须删一半功能上线，你更倾向'按 KPI 权重删'还是'按团队谁在乎哪块留'？"
   —— 工作背景的被试用

### J/P 情境题

1. "你的日历 / 待办 app 打开是什么状态？—— 真实的，不是理想状态。"
   —— ★ **J/P 最稳的题，TA 话短时也能答**
2. "旅行你倾向于提前 2 周订好路线，还是到了再看？讲一次最近的旅行。"
   —— 假期话题顺势
3. "你现在桌面上有几个开着的 tab？"
   —— ★ **TA 话短时最灵**。一个数字就给信号
4. "你是会把一本书读完再开下一本，还是手边同时有 3 本书在轮流看？"
   —— 爱读书的被试用
5. "最近一次你临时改计划 —— 是你自己改的还是被别人改的？你当时什么感觉？"
   —— 反面验证 J/P 的弹性度
6. "你宁可早到 20 分钟等人，还是准时踩点？—— 真实情况，不用美化。"
   —— 快速二选一，有 TA 的真实偏好

## 信号识别指引（LLM 内部判定）

**什么样的回答包含信号**：
- 具体场景（时间、地点、人物、动作）
- 用户自己的评判 / 感受
- 一个可引用的短句

**什么样的回答不包含信号**：
- 纯粹的"对啊 / 嗯 / 有时候"
- 用户反问你
- 用户跑题到别的话题

### 简易映射（示例，非穷尽）

**只有当 TA 在讲一个具体生活场景时**，下列关键词才算信号。**单纯"此刻打字话少 / 用了一个文艺词 / 情绪低"不算**——那是风格和状态，不是维度。

| 用户回答关键词 | 算信号的前提（必须有具体场景） | 倾向 |
|---|---|---|
| "回家路上很累 / 想一个人待会" | TA 在讲**一次聚会之后**回家的感受 | I |
| "越热闹越来劲 / 回家还在兴奋" | TA 在讲**一次和人在一起的场合**结束后的能量 | E |
| "我记得那天咖啡很烫" | TA 在复现**一个具体日子的感官细节** | S |
| "我想着 5 年后……" / "突然联想到……" | TA **自发**跳到未来或不相关联系，而不是被你问出来的 | N |
| "我跟 TA 说这事不合理" | TA 在讲**一次真实决策**时用了逻辑词 | T |
| "我先问 TA 那天什么感觉" | TA 在讲**一次真实决策**时先照顾了感受 | F |
| "我列了清单 / 我都写下来了" | TA 在讲**一个此刻存在的可观察物**（日历、便签、待办） | J |
| "看情况 / 到时候再说" | TA 在讲**一次真实的开放态度**，不是当下在敷衍你 | P |

**两个常见的误读要避开**：

- **TA 此刻话少 ≠ I**。TA 可能是累、不擅长打字、或在等你接话。判 E/I 要听**能量反馈**（"那之后你感觉怎么样"），不是**字数**。
- **TA 用了比喻/意象 ≠ N**。那是 TA 的语言风格。判 S/N 要看**注意力落在哪里**——具体画面还是跳跃联想。一个 S 人也可以很文艺地描述眼前的光。

**到收束时**，每一维你都要能讲出 1-2 个具体场景再打高分。讲不出来就按 `CONVERGE.md` 的 "confidence 是老实话" 把 score 放到 0.6 以下——这不是失败，是诚实。
